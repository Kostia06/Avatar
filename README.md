# Avatar
This program uses webcam to show an avatar on a screen using Python with modules: Cvzone and Pygame.

## IMPORTANT

This program needs to see all of your fingers to make sure the commands work

## COMMANDS
1. Thumb and little finger and point them upwords and hold for few or erase everything
2. Index finger  draw on screen
3. Index and middle finger to select colors
4. Use all of you fingers to make the colors go different directions or stop them
5. Four fingers to activate gravity mode and with your index would be the gravity point or stop them
